Introduction

This web interface can allow HUDS and Green Harvard to model and quantify the Scope 3 Carbon Emissions of HUDS, resulting in HUDS having direct control over their carbon footprint.

This web interface will provide the staff of the dining hall with information on how to calculate and model purchasing methods. It will include different approaches for determining carbon footprints, such as the average-data method and the hybrid method, and the user can customize this interface by choosing and comparing the results of different methods to allow a more holistic approach to measuring HUDS carbon emissions. Also, it will highlight items with the largest output and include suggestions and recommendations on how to reduce them.

    Table highlited in grey are the ones applicable to HUDS, so the web interface will only calculate those fields

    The web interface has three basic functions: add or edit reports, edit database and edit data. The black sidebar on the left can direct the user to any of the basic functions.

    Add report allows the users to calculate scope 3 carbon footprints and the smaller and indented report 1 is a recently created report template. Every time the user adds a new report, the new report will show below "Add Report".

    Edit Database allows the web interface to obtain its emission factors. The user can edit/update the database. "Life Cycle, Weights, Items, Traveling Methods" are databases used for the hybrid method.

    Edit Data allows user to edit the previously uploaded purchasing and waste reports. The reports will be updated according to the uploaded data.

    Toggle Menu is a feature that allows the sidebar to be hidden.
